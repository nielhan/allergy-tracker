데이터 삭제 안내

앱 이름: 알러지한알 (Daily Allergy Bite) · 개발자: monko

← 홈으로

"알러지한알"은 로그인 없이 모든 데이터를 사용자의 기기에만 저장하는 앱입니다. 외부 서버에 개인 식별 데이터를 보관하지 않으므로 별도의 삭제 요청 양식이 필요하지 않으며, 사용자가 직접 기기에서 즉시 삭제할 수 있습니다.

1. 삭제되는 데이터 유형

  • 매일의 복용 기록 (날짜·용량·상태·메모)
  • 치료 단계 설정 (시작일·1회 용량·단위)
  • 알림 시간 설정
  • 선택 언어 등 앱 환경설정
  • Download/AllergyTracker/ 폴더의 자동 CSV 백업 파일

2. 전체 데이터 삭제 방법

방법 A — 앱 제거 (가장 간단, 권장)

  1. Android 홈 화면에서 "알러지한알" 앱 아이콘을 길게 누름
  2. "앱 정보" 또는 "제거" 선택 → "제거" 확인
앱 제거 시 기기 내부 저장소의 모든 기록 즉시 삭제됩니다. 단 Download/AllergyTracker/ 폴더의 CSV 백업은 공용 저장소라 남을 수 있어요. 아래 "방법 C" 참고.

방법 B — 앱 데이터만 삭제 (앱은 남기고 기록만 초기화)

  1. Android 설정 → 앱 → 알러지한알 → 저장공간
  2. "데이터 삭제" 선택 → 확인

방법 C — CSV 백업 파일 수동 삭제

  1. 기본 파일 앱(내 파일 / Files by Google) 실행
  2. 내부 저장공간/Download/AllergyTracker/ 폴더 이동
  3. 해당 폴더 전체 삭제

3. 광고 ID 삭제·재설정

앱은 Google AdMob 을 통해 배너 광고를 노출하며, 이때 기기의 광고 ID(AAID)가 Google 에 전송됩니다. 광고 ID 재설정은 Android 시스템에서 직접 수행합니다:

  1. Android 설정 → 개인정보 (또는 보안 및 개인정보) → 광고
  2. "광고 ID 재설정" 또는 "광고 ID 삭제" 선택

광고 ID 관련 상세 내용은 Google 광고 정책 참고.

4. 보관 기간

  • 앱 내부 데이터: 사용자가 앱을 제거하거나 데이터 삭제를 수행하면 즉시 삭제되며 추가 보관 기간은 없습니다.
  • CSV 자동 백업: 최근 7개만 기기에 유지, 8개째부터는 앱이 자동 정리.
  • 외부 서버 보관: 해당 없음 — 앱은 사용자 기록을 외부로 전송하지 않습니다.
